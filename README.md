# Retail Sales Optimization Analysis

## Overview
This project analyzes Zara retail product data to understand how pricing,
promotion, and product placement affect sales performance.

## Business Questions
- Do promotions improve sales?
- Which store positions generate higher sales?
- How does price affect demand?
- Which products generate the most revenue?

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly

## Key Findings
- End-cap products showed the highest average sales.
- Promotions increased volume but not always revenue.
- Mid-priced products performed better than premium products.
- A small group of products generated most revenue.

## Recommendations
- Move top-selling products to premium store positions.
- Use targeted promotions for mid-priced products.
- Review expensive low-performing products.
